## 🎨 Logo 图标

### 位置
- **代码位置**: `src/components/logo.tsx`
- **使用位置**: 
  - 首页导航栏 (`src/app/page.tsx`)
  - 使用指南页面导航栏 (`src/app/guide/page.tsx`)

### 当前实现
- **格式**: SVG（内嵌在 React 组件中）
- **尺寸**: 默认 32x32 像素（可通过 `size` 属性调整）
- **设计**: 黑色圆角矩形背景 + 白色文档图标 + 转换箭头 + Word 标识

### 替换方法

#### 方法 1: 修改 SVG 代码（推荐）
直接编辑 `src/components/logo.tsx` 文件中的 SVG 路径。

#### 方法 2: 使用图片文件
1. 准备你的 logo 图片（建议 PNG，透明背景）
2. 将图片放在 `public/` 目录下，例如 `public/logo.png`
3. 修改 `src/components/logo.tsx`:

```tsx
export function Logo({ className = '', size = 32 }: LogoProps) {
  return (
    <img 
      src="/logo.png" 
      alt="md2word" 
      width={size} 
      height={size}
      className={className}
    />
  );
}
```

## 🖼️ 模板预览图

### 位置
- **目录**: `public/templates/previews/`
- **配置文件**: `src/config/templates.ts`

### 文件列表

| 文件名 | 对应模板 | 配置字段 | 用途 |
|--------|---------|---------|------|
| `通用样式.svg` | 通用样式模板 | `style-default` | 展示通用 Word 文档样式预览 |
| `公文模板.svg` | 公文模板 | `official-document` | 展示公文格式预览 |
| `论文模板.svg` | 论文模板 | `academic-paper` | 展示论文格式预览 |

### 当前实现
- **格式**: SVG（矢量图，可缩放）
- **尺寸**: 400x300 像素
- **设计**: 简化的文档布局示意图

### 配置对应关系

在 `src/config/templates.ts` 中，每个模板的 `previewImage` 字段指向预览图：

```typescript
{
  id: 'style-default',
  name: '通用样式',
  previewImage: 'templates/previews/通用样式.svg', // ← 这里
  // ...
}
```

**路径说明**:
- `templates/previews/通用样式.svg` 实际对应 `public/templates/previews/通用样式.svg`
- Next.js 会自动将 `public/` 目录下的文件映射到网站根路径

### 替换方法

#### 方法 1: 替换 SVG 文件（保持文件名）
1. 准备你的预览图（建议 PNG 或 JPG，400x300 像素）
2. 保持文件名不变，直接替换 `public/templates/previews/` 目录下的对应文件
3. 如果使用 PNG/JPG，需要同时更新 `src/config/templates.ts` 中的文件扩展名

**示例**:
```bash
# 替换通用样式预览图
# 将你的图片命名为 "通用样式.png" 并放到 public/templates/previews/
```

然后更新配置：
```typescript
previewImage: 'templates/previews/通用样式.png', // 改为 .png
```

#### 方法 2: 使用新文件名
1. 准备你的预览图
2. 将图片放到 `public/templates/previews/` 目录
3. 更新 `src/config/templates.ts` 中的 `previewImage` 字段

**示例**:
```typescript
{
  id: 'style-default',
  name: '通用样式',
  previewImage: 'templates/previews/my-custom-preview.png', // 新文件名
  // ...
}
```

### 预览图要求

- **推荐尺寸**: 400x300 像素（宽高比 4:3）
- **推荐格式**: PNG（支持透明背景）或 JPG
- **文件大小**: 建议 < 200KB，确保加载速度
- **内容建议**: 
  - 展示模板的实际效果（Word 文档截图）
  - 或使用简洁的示意图说明模板特点

## 🔄 替换方法总结

### Logo 替换步骤

1. **准备图片**: 制作 32x32 或更大的 logo（建议 2x 分辨率，即 64x64）
2. **选择方式**:
   - **SVG**: 编辑 `src/components/logo.tsx`
   - **图片文件**: 放到 `public/logo.png`，修改组件代码
3. **测试**: 刷新页面查看效果

### 模板预览图替换步骤

1. **准备图片**: 制作 400x300 的预览图
2. **放置文件**: 放到 `public/templates/previews/` 目录
3. **更新配置**: 修改 `src/config/templates.ts` 中的 `previewImage` 字段（如需要）
4. **测试**: 刷新页面，在模板选择区域查看预览图

## 📝 注意事项

1. **文件命名**: 
   - 使用中文文件名时，确保文件系统支持
   - 建议使用英文文件名避免编码问题

2. **路径格式**: 
   - 在 `previewImage` 中，路径从 `public/` 目录开始，不需要 `/` 前缀
   - 例如：`templates/previews/通用样式.png` ✅
   - 错误：`/templates/previews/通用样式.png` ❌

3. **缓存问题**: 
   - 替换图片后，浏览器可能缓存旧图片
   - 解决方法：强制刷新（Ctrl+F5）或清除浏览器缓存

4. **SVG vs 图片**: 
   - SVG：矢量图，可缩放，文件小，但设计复杂
   - PNG/JPG：位图，文件可能较大，但可以展示真实效果
